1. **`ec7fb3f` feat(highlight)** — Search-needle bolded inside clip
   previews + detail body. New `highlightHtml()` helper in util.ts
   double-escapes (body and needle both) before wrapping matches in
   `<mark class="match-hl">` so it's injection-safe. Wired into
   `renderClip`, `openDetail`, redact/unredact refresh paths, and the
   reveal-once banner. Operator tokens stripped via `parseQuery().freeText`
   so `kind:image foo` only highlights "foo". 9/9 sanity cases pass.

2. **`f01d4fc` feat(history)** — Recent search history as muted
   dashed-border ghost chips below the saved-searches strip. Captures
   debounce ~900ms (no per-keystroke writes), cap 5, dedup against
   saved searches so the user never sees a query in two strips. New
   `listSearchHistory/pushSearchHistory/clearSearchHistory` in db.ts
   using the same single-meta-row pattern as saved_searches (no schema
   bump). Inline "×" clears the whole list for privacy.

3. **`6a033f0` feat(cheatsheet)** — `?` opens a 2-column glass-card
   modal with every shortcut + search operator, grouped into Navigate /
   Clip actions / Detail view / Search operators. Esc, backdrop click,
   and × all close. `<kbd>` styling reusable everywhere. Cheatsheet
   check sits at the very top of the keydown listener so it works
   regardless of which sub-panel is open. Footer hint promotes
   "? cheatsheet" first.

4. **`82a97a0` feat(storage)** — Replaced the single quota gradient bar
   in Settings with a stacked segment bar split by text / images /
   links / OCR / trash, plus a colored-dot legend with byte totals.
   Uses raw `flex: <bytes>` so widths are correct without JS math;
   tooltips on each segment show label + bytes + percent. Quota row
   degrades gracefully when navigator.storage.estimate has no quota.
   Bottom foot-line shows live + trashed totals so users decide
   "should I empty trash?" at a glance.

5. **`54a2c1a` feat(bulk)** — Select all filtered, three entry points:
   footer accent pill "Select all N" (visible only when filter active
   and nothing selected), new check-icon toggle inside the bulk bar
   (acts as deselect-all when everything's already selected, with
   active accent fill), and `Cmd/Ctrl+A` shortcut (skips inside search
   input to preserve native text-select-all). Additive across filter
   changes — selecting in `host:bank.com` then switching to
   `host:other.com` keeps the bank selection. Cheatsheet updated.

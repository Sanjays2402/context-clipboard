## Features

1. **Cmd/Ctrl+K command palette** — `1c3fb04`
   - New `src/lib/palette.ts` with pure fuzzy matcher (substring +
     acronym + sequential scoring, hit-offset bolding). Unit-testable
     without DOM; 11/11 sanity (`.cron-state/sanity-palette.mjs`).
   - Actions grouped Navigate / Capture / Privacy / Filter / Bulk /
     Sort / Export. Contextual `available` flags hide irrelevant rows
     (Empty Trash when empty, Clear Selection when no selection, etc.).
   - New helpers: `pinAllFiltered()` (bulk pin/unpin without manual
     multi-select first), `clearAllFilters()` (one-shot reset),
     `applyKindFilter()` (single source of truth used by filter
     buttons + palette).
   - Cheatsheet now mentions ⌘/Ctrl+K.

2. **Footer sort dropdown** — `3d2bce9`
   - New `src/lib/sort.ts`. Modes: `recent` (default) / `oldest` /
     `hits` (most copied) / `size` (largest) / `alpha` (A→Z).
     Pinned ALWAYS float to top regardless of mode. Pure.
     13/13 sanity (`.cron-state/sanity-sort.mjs`).
   - Persisted in IDB meta as `list_sort` via new
     `getListSort`/`setListSort`. Read at boot BEFORE first render.
   - Dropdown picks up an accent border when non-default sort active.
   - Also wired into the palette as a "Sort" group.

3. **Anti-shoulder-surf blur** — `7b57e89`
   - New `blurPreviews` setting (additive, defaults false).
   - Body-level `blur-on` class triggers a 4px CSS blur on previews,
     thumbs, tags, detail body, OCR text, nearby context, template
     preview, trash preview. Hover or `.active` (arrow-nav) snaps
     back to clarity.
   - Accent header badge ("blurred — hover to read") so the mode is
     never silently active.
   - Live-preview via `applyBlurMode()` — toggle in settings shows
     effect without leaving the panel. Cmd+K quick-toggle.

4. **Import dedup by hash** — `57d3fa4`
   - `importAll` was id-only. Now two-tier:
     1) id collision → `skippedId`
     2) hash collision (new id, same content) → MERGE into live row:
        hitCount summed, lastSeenAt advanced, tags unioned, pin
        stickier
     3) genuinely new → `imported`
   - Hash index built once, updated as inserts happen (so internal
     dups within the same file dedup correctly too).
   - Popup toast surfaces dedup outcomes only when they happened.
   - 14/14 sanity against an in-process IDB shim
     (`.cron-state/sanity-import-dedup.mjs`).

5. **Omnibox `cc` keyword** — `74201a6`
   - Type `cc` + Tab + text in URL bar → captures as `kind:text`,
     source "Omnibox note", tagged `omnibox`. Sub-second capture
     path that bypasses the popup.
   - `onInputChanged` surfaces matching past omnibox notes as
     autocomplete rows ("Recall: <preview>"); selecting bumps the
     hitCount/lastSeenAt so it floats to top of popup next open.
   - `escapeOmnibox()` sanitizes descriptions (Chrome's omnibox
     treats them as XML-ish — naked `<` / `&` break the row).
   - PII auto-redact respected via the existing ingest path.
   - New manifest `omnibox` block on chrome + firefox.
   - Cheatsheet "Address bar" section added.

## Defers / notes

- Snippet templates intentionally do NOT support a `{{prompt:label}}`
  interactive token (no input dialog at copy time). Roadmap idea, not
  in scope for this tick — current tokens are all derivable from
  context, which keeps copy a single sync write.
- TTL UI deliberately uses durations-from-now instead of trying to
  "show current selection" in the dropdown. Selecting "Never" clears,
  selecting any preset replaces. Simpler than a calendar picker and
  hits 95% of real use ("expire this in a day").
- Forget host routes through trash so deletion is recoverable. If a
  user really wants something gone NOW, "Empty trash" is one click
  further. Privacy posture: data still local; the user retains
  control over the irreversible step.
- `confirm()` dialogs removed for single + bulk delete; Undo toast is
  strictly better. Kept for "Delete everything", "Forget host"
  (concrete pre-flight count makes it actionable), and "Empty trash".

## Shipped

1. `def8a6c` **feat(search): is:hostlocked** — cross-store join
   (site_rules × clips) under first-match-wins semantics. Distinct
   lens from `is:locked` (per-clip bit) — answers "is this from a
   site I've protected by RULE?" rather than "did I lock THIS
   clip?". Pure host-locked module with closure-scoped per-host
   verdict cache; popup wires single-RPC rules-refresh per
   render() so the predicate is fresh every paint (no stale rule
   decisions). applyQuery gains hostLockedPredicate opt that falls
   open when unsupplied (test-friendly). Pairs orthogonally with
   `is:locked` (alignment) and `is:unlocked` (drift). 24/24
   sanity.

2. `02fe975` **feat(search): is:notelonger:N / is:noteshorter:N**
   — note-length filters with strict comparison (>N / <N — the
   boundary belongs to neither, partitioning the space at any
   chosen N). Both implicitly require is:noted (no-note clips
   never match — operators are about note LENGTH, not absence).
   Strict-integer parser rejects decimals/signs/empty-tail → free
   text (typo visibility). AND-semantics composition yields a
   band-pass filter. Cmd+K defaults: 120 (essay candidates) / 30
   (sticky-note tag-promotion candidates). 22/22 sanity.

3. `28a02fc` **feat(note-composer): pre-fill from active tab
   title** — replaces blank textarea with "Captured from <title>"
   stem. Pure note-prefill module: one-pass site-suffix strip
   (preserves legitimate subtitles via 30-char cap + terminal-
   punctuation guard), 80-char cap with word-boundary ellipsis,
   host fallback (chrome:// / file:// rejected). No-clobber guard
   for re-opened mid-edit composer drafts. api.tabs.query try/
   catch wrapped for chrome:// scope. No auto-select-on-focus
   (anti-pattern that wipes drafts on first keystroke). 35/35
   sanity.

4. `53b42cf` **feat(palette): "Note every clip from active host"**
   — third leg of host-triage trio (pin/lock/note). Pure host-
   note module mirrors host-lock shape exactly. planHostNote uses
   SAME sanitizeClipNote pipeline as detail editor + bulk-bar
   (single source of truth across all three entry points). Apply
   path re-reads live store at click time + counts currently-noted
   matches so prompt label warns "(N will be replaced)" up front;
   planning gates the IDB loop so idempotent re-runs no-op
   cleanly. 28/28 sanity.

5. `056cbd8` **feat(detail): send-to "Copy note as Markdown"** —
   wraps clip's free-form note as a `> ` blockquote. Pure note-
   markdown module: CRLF/bare-CR normalisation, outer-blank-strip,
   each-line `> ` prefix (empty inner lines become `>` for
   paragraph breaks), nested-quote pass-through (`> > existing`).
   Row HIDES (not greys) when un-noted so menu stays tight.
   SendableClip type extended with optional note; popup passes
   c.note in BOTH buildSendActions call sites so menu-render +
   click-dispatch gates stay consistent. Bumped existing
   sanity-send-to count 14→15. 28/28 new sanity.

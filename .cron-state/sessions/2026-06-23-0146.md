## Notes
- The detail-note textarea ships as ALWAYS-visible (not conditional like
  the other meta rows) so the "add a note here" affordance is obvious
  even on noteless clips. Different from Locked/Template/OCR which
  surface STATE, not invite input.
- `is:noted` ships in the same tick as the note feature so the operator
  is immediately useful — a noted clip is searchable from the moment
  it's saved.
- `findLiveRecaptureForTrash` uses strict > -Infinity tie-break, which
  means all-unstamped matches return null instead of arbitrarily picking
  the first. Trade-off: a freshly-imported live clip with no lastSeenAt
  won't surface as a re-capture until it's re-copied (sets lastSeenAt).
  Acceptable — that's a vanishingly rare case.
